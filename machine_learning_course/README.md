
---

## 📖 Each Notebook Contains

Every notebook follows the **same structure**:

1. **📚 THEORY** — Concept explanation with formulas and intuition
2. **🔬 PRACTICAL** — Short, focused code cells to run immediately
3. **🏋️ Try It Yourself** — A cell where YOU change values and experiment
